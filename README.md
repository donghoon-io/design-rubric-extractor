# Design Rubric Extractor

A Streamlit-based application for tailoring design rubrics.  
This repository contains a Streamlit app defined in `app.py`.

---

## Requirements

- Python 3.8 or higher
- pip (Python package manager)

---

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/design-rubric-extractor.git
   cd design-rubric-extractor
    ```

2. **(Optional) Create a virtual environment**

   ```bash
   python -m venv venv
   source venv/bin/activate   # macOS/Linux
   venv\Scripts\activate      # Windows
   ```

3. **Install Streamlit**

   ```bash
   pip install streamlit
   ```
---

##  Running the App

From the root of the repository, run:

```bash
streamlit run app.py
```

The app will open in your default browser.
If it does not open automatically, visit:

```
http://localhost:8501
```
